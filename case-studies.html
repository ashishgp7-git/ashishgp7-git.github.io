Zabbix High Availability with Galera Cluster

Production monitoring environment · High availability requirement · Zero data loss

Problem:

Monitoring was running on a single Zabbix server, creating a single point of failure. Any outage resulted in loss of visibility and operational risk.

Solution:

  • Designed and implemented Zabbix High Availability architecture
  • Configured MariaDB Galera Cluster for real-time database replication
  • Implemented Pacemaker and Corosync for automated failover
  • Configured Virtual IP (VIP) for seamless access

Tools & Technologies:

Zabbix, MariaDB Galera, Pacemaker, Corosync, Linux

Result:

Achieved zero data loss monitoring with automatic failover and significantly improved reliability for production operations.

Azure Disaster Recovery using Azure Site Recovery (ASR)

Business-critical Azure workloads · Disaster recovery & compliance driven

Problem:

Business-critical Azure VMs lacked a tested disaster recovery strategy, exposing the organization to downtime risks.

Solution:

  • Designed Azure DR architecture with compliance considerations
  • Configured Azure Site Recovery for multiple VMs
  • Implemented Azure Backup and restore validation
  • Documented DR runbooks and performed test failovers

Tools & Technologies:

Azure VMs, Azure Site Recovery, Azure Backup, VNets, NSG

Result:

Reduced recovery time objectives (RTO) significantly and provided business assurance with tested disaster recovery capability.

AWS Infrastructure & PCI DSS Compliance Audit

Production AWS environment · External audit & regulatory compliance

Problem:

AWS environment required PCI DSS compliance validation for external audit and regulatory requirements.

Solution:

  • Reviewed AWS VPC, EC2, IAM, Security Groups, and network architecture
  • Validated access control, logging, and patching mechanisms
  • Prepared audit-ready evidence and documentation
  • Implemented remediation recommendations

Tools & Technologies:

AWS EC2, VPC, IAM, SSM, Security Groups, PCI DSS controls

Result:

Successfully supported PCI DSS audit with compliant architecture and clear, verifiable security evidence.
